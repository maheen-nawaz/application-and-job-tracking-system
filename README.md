# Jobify Pro 💼

Jobify Pro is a full-stack job portal web application designed to connect recruiters and job seekers through an organized hiring platform.

The system allows users to manage job applications, communicate with recruiters, and track hiring progress.

## Features ✨

### Job Seekers
- Create and manage profiles
- Browse available jobs
- Apply for jobs
- Track application status
- View application history

### Recruiters
- Post job opportunities
- Manage applicants
- Review applications
- Communicate with candidates

### Admin Panel
- Manage users
- Monitor system activities
- Maintain platform control

## How to Run ▶️

1. Clone the repository.

2. Install dependencies:

```bash
npm install
```

3. Configure database connection.

4. Start the server:

```bash
npm start
```
